Placeholder Image for Movie Posters

This document provides information about using placeholder images for movie posters when the actual poster is not available from the TMDB API.

Example Placeholder

How to Use

You can use this data URI directly in your HTML or templates when a movie poster is not available:

<img src="data:image/svg+xml;charset=UTF-8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20width%3D%22300%22%20height%3D%22450%22%20viewBox%3D%220%200%20300%20450%22%3E%3Crect%20fill%3D%22%23333%22%20width%3D%22300%22%20height%3D%22450%22%2F%3E%3Ctext%20fill%3D%22%23fff%22%20font-family%3D%22Arial%2C%20sans-serif%22%20font-size%3D%2224%22%20text-anchor%3D%22middle%22%20x%3D%22150%22%20y%3D%22225%22%3ENo%20Poster%20Available%3C%2Ftext%3E%3C%2Fsvg%3E" alt="No Poster Available">

Or in your Flask templates, you can use a conditional to show either the actual poster or the placeholder:

{% if movie.poster_url %}
    <img src="{{ movie.poster_url }}" alt="{{ movie.title }} Poster">
{% else %}
    <img src="data:image/svg+xml;charset=UTF-8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20width%3D%22300%22%20height%3D%22450%22%20viewBox%3D%220%200%20300%20450%22%3E%3Crect%20fill%3D%22%23333%22%20width%3D%22300%22%20height%3D%22450%22%2F%3E%3Ctext%20fill%3D%22%23fff%22%20font-family%3D%22Arial%2C%20sans-serif%22%20font-size%3D%2224%22%20text-anchor%3D%22middle%22%20x%3D%22150%22%20y%3D%22225%22%3ENo%20Poster%20Available%3C%2Ftext%3E%3C%2Fsvg%3E" alt="No Poster Available for {{ movie.title }}">
{% endif %}

Alternative Approach

Instead of using a data URI, you can also use a div with CSS styling as we've implemented in our templates:

{% if movie.poster_url %}
    <img src="{{ movie.poster_url }}" alt="{{ movie.title }} Poster">
{% else %}
    <div class="no-poster">No Poster Available</div>
{% endif %}

With the following CSS:

.no-poster {
    width: 100%;
    height: 100%;
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 20px;
}

This approach is already implemented in our templates and is the recommended way to handle missing posters.
