# Image Files for Movie Website

This directory should contain the following image files:

1. `placeholder.jpg` - A placeholder image for movie posters
2. `404.png` - An image for the 404 error page
3. `500.png` - An image for the 500 error page

Please add these images manually or replace them with your own images.

## Image Specifications

- **placeholder.jpg**: Recommended size is 300x450 pixels (typical movie poster ratio)
- **404.png**: Recommended size is 400x400 pixels
- **500.png**: Recommended size is 400x400 pixels

You can use any image format (JPG, PNG, GIF, etc.) as long as you update the references in the HTML templates accordingly.
