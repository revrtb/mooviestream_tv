Mobile App Downloads Directory
===============================

Place your Android APK files here with the naming pattern: Mooviestream-X.X.X.apk

Examples:
- Mooviestream-1.0.2.apk
- Mooviestream-1.0.3.apk
- Mooviestream-1.1.0.apk

The system will automatically detect and serve the latest version.

The file will be accessible at: /download/android

File location: static/apps/Mooviestream-X.X.X.apk

Note: The latest version (highest version number) will be automatically served when users click download.
